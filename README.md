# Practica-de-recetario-BOOTSTRAP
Proyecto colaborativo desarrollado con *HTML, CSS y Bootstrap 5* por un equipo de 8 integrantes.

##  Descripción
Recetario digital donde cada integrante aporta sus propias recetas favoritas, 
organizadas en tarjetas interactivas con carrusel de imágenes.

## Tecnologías usadas
- HTML5
- CSS3
- Bootstrap 5.3
